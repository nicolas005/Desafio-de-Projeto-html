h1, h2, h3, h4, h5, h6

(cabeçalhos de nível 1 a 6): Essas tags são usadas para criar cabeçalhos com diferentes níveis de importância. h1 é o cabeçalho mais importante, enquanto h6 é o menos importante. Elas são usadas para organizar o conteúdo da página.

p

(parágrafo): Essa tag é usada para definir um parágrafo de texto em HTML. Ela cria uma quebra de linha antes e depois do texto dentro dela.

exemplo

small

(pequeno): Essa tag é usada para definir texto menor que o tamanho normal. Geralmente, é usada para texto de rodapé ou informações de menor destaque.

exemplo

i

(itálico): Essa tag é usada para aplicar estilo de texto itálico ao conteúdo dentro dela.

exemplo

u

(sublinhado): Essa tag é usada para aplicar sublinhado ao texto dentro dela.

exemplo

strong

(destaque): Essa tag é usada para enfatizar o texto, tornando-o mais forte ou importante. Geralmente, o texto dentro dela é exibido em negrito.

exemplo

a

(âncora): Essa tag é usada para criar hiperlinks. Ela permite que os usuários cliquem em um texto ou imagem para navegar para outra página da web.

exemplo

hr

(linha horizontal): Essa tag é usada para inserir uma linha horizontal na página, separando o conteúdo.

sub

(subscrito): Essa tag é usada para criar texto subscrito, geralmente usado para fórmulas químicas ou matemáticas.

exemplo

sup

(sobrescrito): Essa tag é usada para criar texto sobrescrito, geralmente usado para expoentes ou notas de rodapé.

exemplo

blockquote

(citação em bloco): Essa tag é usada para destacar um bloco de texto como uma citação, geralmente com um recuo ou estilo diferente.

exemplo

font

(fonte): Essa tag costumava ser usada para definir a fonte e o tamanho da fonte do texto, mas seu uso foi desencorajado em favor do CSS (Cascading Style Sheets).

del

(texto deletado): Essa tag é usada para indicar texto que foi removido ou deletado de uma versão anterior.

exemplo

abbr

(abreviação): Essa tag é usada para indicar uma abreviação ou acrônimo e pode conter um atributo "title" que fornece a expansão completa da abreviação quando o usuário passa o mouse sobre ela.
